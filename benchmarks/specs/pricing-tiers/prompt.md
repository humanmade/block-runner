# Pricing tiers (feature cards)

A pricing section: an H2 "Simple pricing", then three plan cards in a row. Each card is a
self-contained box (border/shadow) holding, in order: a plan name (H3), a price, a feature
list, and a button.

- **Starter** — "$0" — features: "1 project", "Community support" — button "Choose Starter"
- **Pro** — badge "Most popular", "$29" — features: "Unlimited projects", "Priority support",
  "CI integration" — button "Choose Pro"
- **Scale** — "Custom" — features: "SSO", "Dedicated support" — button "Contact sales"

Each card is visually a distinct container. Output one self-contained HTML `<section>`;
semantic HTML, your own CSS/classes.
